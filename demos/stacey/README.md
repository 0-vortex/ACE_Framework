# Stacey
This a work-in-progress demo prototype for the ace framework.

We'll start as a simple chatbot and expand from there.

## Backend
- `cd backend`
- `cp .env.example .env` and set the OPENAI_API_KEY
- `pip install -r requirements.txt`
- `python app.py`

## Frontend
- `cd frontend`
- `npm install`
- `npm run dev`
